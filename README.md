earthquakes
===========

List of earthquakes in Nicaragua

# Clone repo and go to directory

    $ git clone https://github.com/apinic/earthquakes.git
    $ cd earthquakes

# Install dependencies

    $ npm install

# Run

    $ node index.js

# Open browser

    http://localhost:5000

## Built with

* [hapiJS](http://hapijs.com/)
* [ineter](https://www.npmjs.com/package/ineter)
